# Neon Apps Academy

Selamlar! 👋 Bu repoda, Neon Academy trainee programı sürecinde sıfırdan inşa edip tamamladığım **47 farklı Flutter görevi (task)** yer almaktadır.

Bu arşivin temel amacı; temel UI bileşenlerinden gelişmiş kaydırma mimarilerine kadar uzanan süreçte, Clean Code prensiplerine ve modern Flutter standartlarına uygun geliştirme becerilerimi sergilemektir.

## 🛠️ Süreç Boyunca Neler Geliştirdik?

47 görevlik bu yolculukta sadece ekrana widget koymakla kalmadık, performansı ve kullanıcı deneyimini (UX) ön planda tuttuk. Öne çıkan bazı odak noktalarım:

* **Esnek ve Dinamik Arayüzler:** `CustomScrollView`, `SliverAppBar` ve `SliverList` kullanılarak tasarlanan yüksek performanslı, kaydırdıkça şekil değiştiren haber akışları.
* **Etkileşimli Bileşenler:** `CupertinoContextMenu` ile şık iOS menüleri, dinamik `GridView` yapıları ve anlık veri filtreleyen arama çubukları (Search Bar).
* **Veri Gösterimi:** Özelleştirilmiş `DataTable` yapıları ve anlık yenileme (Pull-to-Refresh) mekanizmaları.
* **Dış Bağlantılar:** `url_launcher` entegrasyonu ile uygulama içinden tarayıcı ve uygulama mağazası yönlendirmeleri.
